Ruta SV v4 - Google Maps Upgrade

1. Run maps-setup.sql in Supabase SQL Editor.
2. Deploy the full folder to Netlify.
3. Open any place details page and click "Open in Google Maps".
4. In Admin, edit a place and update the new "Google Maps URL" field when you want a more exact Maps link.

If you do not provide a Google Maps URL, the app automatically creates a Google Maps search link from the place name and address.
